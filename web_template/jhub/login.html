Acknowledgement

What is UIC-STAT Bayes Cluster?

UIC-STAT Bayes Cluster, or USBC, is the first generation high-performance computing cluster, which provides high-performance computing services for both students and teachers in Statistics Program. The user can run the pre-built software, such as GCC, MATLAB, Anaconda etc. on the cluster. The cluster brings the power of efficiency to groups of users.It gives users access to computational environments and resources without burdening the users with enviornment maintenance - and it can get their work done in their own workspaces on shared resources, which can be managed efficiently by system administrators. For more information, please check our wiki

What is Web Portal

In USBC, we use JupyterHub as the web Portal and the user can use a web-based VS Code as their GUI. Previously, the cluster user needs to access the Cluster via SSH, and most of the users are not familiar with CLI. With this web-based system, the user can intuitively access to the system can do the coding, check files, and even submit, check and stop their SLURM jobs.

In the future, we may add an interactive cluster for GUI-based softwares (e.g. VNC, MATLAB, RStudio).

Who can use this service?

For now, only UIC-STAT students and researchers can apply an account, if you fullfil the requirement, after completing the application form, we will create the account for you within three weekdays.
